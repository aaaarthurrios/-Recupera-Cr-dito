# Análise de Recuperação de Crédito

Projeto em Python para análise de clientes inadimplentes com dashboard interativo.

## Tecnologias
- Python
- Pandas
- Streamlit

## Como rodar
```bash
pip install -r requirements.txt
streamlit run dashboard/app.py
```
